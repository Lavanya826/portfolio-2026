# Lavanya Warang — Portfolio

Pure HTML / CSS / JavaScript portfolio site. No build step, no frameworks — just open `index.html` in a browser.

## Structure
```
portfolio/
├── index.html      → all page content & sections
├── style.css       → theme, layout, animations
├── script.js       → loader, cursor, particles, scroll reveals, form, carousel
└── assets/
    ├── images/      → put project screenshots / photo here
    └── icons/       → put any custom icons here
```
